## 🛠️ Troubleshooting & Tips

Here are some common questions and fixes if you have trouble using sound_rec:

- **No sound input detected**  
  - Make sure your microphone or audio device is plugged in and enabled in Windows.  
  - Check sound settings in Windows Control Panel or Settings app to confirm input devices.

- **Audio levels never move**  
  - Try adjusting the microphone volume in Windows settings.  
  - Make sure no other applications are blocking or using the audio device exclusively.

- **Recording plays back with noise or static**  
  - Try moving the microphone away from noisy electronics.  
  - Use a different microphone if possible to see if the problem persists.

- **Program crashes or freezes**  
  - Restart your computer and run sound_rec again.  
  - Update your graphics card drivers to the latest version to support OpenGL well.

- **Cannot save file or save dialog does not appear**  
  - Check if you have permission to write files in the chosen folder.  
  - Try saving to a different folder like Desktop or Documents.
